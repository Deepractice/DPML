# DPML 示例

本目录包含DPML核心模块的使用示例，帮助你理解如何使用DPML库进行文档处理。

## 示例列表

### 处理器基本使用 (processor-basic-usage.ts)

展示如何使用DPML处理器模块处理文档，包括：
- 创建和配置DefaultProcessor
- 注册自定义访问者
- 处理简单文档
- 引用解析
- 错误处理

运行示例：
```bash
# 从项目根目录
npx ts-node examples/processor-basic-usage.ts
```
